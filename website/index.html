Zero-Install Deployment Guide

NexusDown is packaged as a standalone portable application. Set it up on any Windows client machine in 3 simple steps.

1

Download ZIP Package

Download NexusDown.zip. This archive contains the compiled program, model weights, and template configuration file.

2

Extract Package Contents

Extract the compressed zip to your target folder on your client PC (e.g. your desktop or user directory).

3

Configure & Launch

Double-click `NexusDown.exe` to run. Open the Credentials Menu drawer to input Member IDs and passwords to save your local configuration settings.
